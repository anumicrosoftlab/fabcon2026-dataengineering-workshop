# 🙌 Fabric Spark Data Engineering Workshop

Join this hands-on workshop to learn how a **Microsoft engineering team** uses **Fabric Spark** to tackle real-world data engineering challenges.

## 🎯 What You'll Learn

- 🛠️ Build **scalable data pipelines** with Spark Structured Streaming
- 🤖 Apply **ML augmentation** to your data workflows
- 🏅 Implement an **end-to-end Medallion Architecture** (Bronze → Silver → Gold)

You'll leave with **reusable code assets, patterns, and tips** 💡 to accelerate building your own enterprise-ready data pipelines that can be debugged, deployed, and tested. 🚀

---

## 🚀 Getting Started

### Step 1 — Create a New Workspace

1. Navigate to [Microsoft Fabric](https://app.fabric.microsoft.com/)
2. In the left nav, click **Workspaces** → **+ New workspace**
3. Give your workspace a meaningful name (e.g., `FabricSparkWorkshop`)
4. Click **Apply** to save

---

### Step 2 — Import the Jumpstart Notebook

1. Download the **`Deploy Jumpstarts.ipynb`** notebook
2. In your new workspace, click **+ New** → **Import notebook**
3. Upload the `Deploy Jumpstarts.ipynb` file
4. Once imported, open the notebook

---

### Step 3 — Run the Notebook on PySpark Kernel

1. In the notebook toolbar, confirm the kernel is set to **PySpark**
   - If not, click the kernel selector and choose **PySpark**
2. Run all cells to **install the labs** on your environment
3. Follow the in-notebook instructions as each cell executes

> ⚠️ **Note:** Ensure your workspace has a Fabric capacity (F-SKU or Trial) attached before running the notebook.

---

### Step 4 — Install Labs via Fabric Jumpstart

Use **Fabric Jumpstart** to install the workshop labs with one-click installation:

| Resource | Link |
|---|---|
| 🌐 Fabric Jumpstart Portal | [jumpstart.fabric.microsoft.com](https://jumpstart.fabric.microsoft.com/) |
| 💻 GitHub Repository | [github.com/microsoft/fabric-jumpstart](https://github.com/microsoft/fabric-jumpstart) |

> 🏆 *Fabric Jumpstart provides high-quality demos, tutorials, and accelerators with one-click installation that just work — experience the full power of Microsoft Fabric.*

---

### Step 5 — Follow the Lab Instructions

1. After installation completes, open the lab notebooks in your workspace
2. Follow the step-by-step instructions within each lab
